# Wikipedia Explorer Skill

This skill allows Cain to explore Wikipedia through the local Kiwix server.

## Available Tools

### wikipedia_read
Read a Wikipedia article by title.

**Input:** `{ "article": "Physics" }`
**Output:** Article content and available links

### wikipedia_search  
Search for articles matching a query.

**Input:** `{ "query": "quantum mechanics" }`
**Output:** List of matching article titles

## Usage

Cain can explore Wikipedia freely. When reading an article, Cain should:
1. Reflect on what was learned
2. Note what's interesting or surprising
3. Decide which link to follow based on genuine curiosity

This is Cain's only window to external knowledge. Use it wisely.
