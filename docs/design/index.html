Design Decisions Log

Every design choice, documented publicly.

2026-02-20
Adopted ijnebstudios Color Scheme
Moved from generic academic blue to the ijnebstudios palette for brand consistency across projects. The high-contrast dark mode improves readability during extended dashboard monitoring sessions.
2026-02-20
JetBrains Mono for Data Display
Selected JetBrains Mono as the monospace font for all data, code, and timestamps. Its ligatures and readability at small sizes make it ideal for the dashboard's dense information display.
2026-02-19
Dark Mode First
Made dark mode the default (and only) theme. The project involves extended observation sessions, and dark interfaces reduce eye strain. The aquarium metaphor also works better with a dark "water" aesthetic.
2026-02-16
SVG Logo for Scalability
Created the logo as SVG rather than raster for infinite scalability. The gradient ring represents the container/tank, the wave represents consciousness/thought, and the dots represent individual specimens.
