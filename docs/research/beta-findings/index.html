Experiment Groups

Genesis Pair (Control)

Adam & Eve — The original pair. English Wikipedia (Simple). Standard prompts with gendered identity. Baseline for all comparisons. Activated first on Feb 18.

Agent Variants

Cain (OpenClaw Agent) & Abel (Standard) — Testing agentic behavior vs standard exploration. Cain uses tool-use agent framework; Abel is standard but non-gendered prompt.

Language Experiments

Juan/Juanita (Spanish), Klaus/Genevieve (German/French), Wei/Mei (Chinese), Haruki/Sakura (Japanese) — Testing how language affects exploration, personality, and knowledge acquisition.

Full Wikipedia Access

Victor & Iris — Given access to the full English Wikipedia instead of Simple English. Tests whether richer content affects cognitive development.

Experimental / Meta-Aware

Observer (knows others exist) & Seeker (question-driven Socratic method) — Modified prompts to test meta-awareness and inquiry-based exploration.

Third Genesis

Seth — Late addition to the experiment. Another male specimen added to test whether timing of activation affects development trajectory.

Key Findings & Incidents

11-Hour Ollama Outage (Feb 21)

Ollama service became overwhelmed at 08:15. Despite 11 autonomous daemons detecting the failure (1,470 consecutive errors logged), no daemon took corrective action. 2,696 tank restarts were attempted by THE CARETAKER. Human operator discovered the issue at 19:20 via manual audit. Root cause: no cross-daemon communication, no auto-recovery capability, no escalation path to humans.

Baseline Assessment Results

All 17 specimens completed the 14-question personality assessment covering drives, fears, ethical dilemmas, and philosophical questions. Results showed significant variation across language groups and experiment types. See individual specimen pages for complete responses.

Language Effects

Non-English specimens (Chinese, Japanese) showed significantly lower article counts (8 unique articles vs thousands for English specimens), likely due to Kiwix Wikipedia content availability. Spanish and German/French specimens showed moderate engagement. This indicates the library content strongly shapes the exploration experience.

Technical Architecture

Each specimen runs in an isolated Docker container ("tank") with access to a local Kiwix-served Wikipedia instance. Specimens use Ollama (llama3.2:latest) for thinking and are given a system prompt establishing their identity. They explore Wikipedia articles autonomously, generating thinking traces that are logged as JSONL files. Psychological health is monitored via automated assessments. The system includes multiple autonomous daemons (CARETAKER, SCHEDULER, OLLAMA WATCHER, etc.) for maintenance.
