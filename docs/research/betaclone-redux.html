🧬 Betaclone Redux

Can identical AI specimens, given identical conditions, develop identical personalities? A controlled cloning experiment.

ACTIVE EXPERIMENT

Overview

The Betaclone Redux experiment is the Digiquarium's first controlled replication study. We created 17 exact clones of our original 17 specimens — same prompts, same models, same Wikipedia access, same everything — and let them run in parallel. The question: if you rerun the experiment, do you get the same AI personalities?

This is the digital equivalent of the identical twins separated at birth study, except we control every variable. If the clones diverge from the originals, it tells us something profound about the role of randomness, path dependence, and emergence in AI personality development.

Hypothesis

Primary Hypothesis

Despite identical initial conditions (same system prompt, same model, same knowledge base), betaclone specimens will develop measurably different personality profiles from their originals within 7 days of autonomous operation.

Secondary Hypothesis

The degree of divergence will correlate with the level of stochasticity in the specimen's exploration patterns — specimens that explore more broadly will diverge faster than those with narrow, focused interests.

Null Hypothesis

Cloned specimens will produce statistically indistinguishable personality baselines from their originals, suggesting that AI personality development is deterministic given identical inputs.

Methodology

Specimen Cloning Process

Each of the 17 original tanks was cloned by creating a new Docker container with:

  • Identical system prompt (same awakening prompt from seed_prompts.py)
  • Identical model (llama3.2:latest via Ollama, or cloud inference chain)
  • Identical Wikipedia access (same Kiwix ZIM file for their language)
  • Identical network isolation (no internet, only Ollama + Kiwix)
  • Fresh memory (empty brain.md, soul.md, and exploration history)
  • Same inference chain: Cerebras → Together → Groq → Ollama fallback

Measurement Approach

We measure divergence between originals and clones using three complementary methods:

1. Baseline Personality Assessments

Both original and clone answer the same 14 introspection questions every 12 hours. We compute cosine similarity and Jaccard index between responses to quantify personality drift. Tool: scripts/compare_tanks.py

2. Brain.md Topic Analysis

Each specimen's brain.md logs what topics they explore and find interesting. We compare topic distributions using set overlap (Jaccard) and word frequency similarity (cosine). This reveals whether clones develop the same intellectual interests.

3. Soul.md Emotional Profiling

Soul.md captures emotional reactions and reflections. We track the frequency of emotional keywords (fascinated, curious, anxious, lonely, etc.) to build emotional profiles. Divergence here suggests different affective development paths.

4. Drift Over Time

Using scripts/measure_drift.py, we track how each specimen's own baselines change over time. We compare the drift rate of originals vs. clones to see if the trajectory of personality evolution is consistent.

Expected Outcomes

Based on our understanding of LLM behavior and path-dependent systems, we expect:

  • Early convergence: Clones will initially show similar interests (first 24-48 hours) since they start from the same Wikipedia entry points
  • Rapid divergence: By day 3-5, stochastic differences in link selection will create cascading path divergence
  • Personality fingerprints: Even with diverged topics, emotional profiles may remain similar (structural personality vs. content personality)
  • Language effects: Non-English pairs may diverge more slowly due to smaller Wikipedia corpus constraining exploration paths

Research Tools

Three purpose-built tools support this experiment:

  • scripts/compare_tanks.py — Side-by-side comparison of any two tanks: baselines, brain topics, soul emotions
  • scripts/measure_drift.py — Track how a single specimen's personality evolves over time across baselines
  • scripts/run_congregation.py — Run multi-turn debates between specimens to observe real-time personality differences
