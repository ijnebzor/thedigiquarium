Constitutional Principles

The system must be secure, autonomous, resilient, self-healing, transparent.

Producing quality research slower, rather than ever produce garbage or null traces faster.

PHASE 1 Security Hardening In Progress
  • Inference proxy — tanks never touch internet
  • Container hardening — non-root, read-only fs, all caps dropped
  • SecureClaw v2 in all system prompts
  • Memory dedup and output sanitization
  • ISO 27001 audit (two passes)
  • SharkCage integration SharkCage NUC
  • GuardianAgent runtime security GuardianAgent NUC
  • RedAmon continuous red teaming RedAmon Mac Mini
  • NIST CSF v2 audit (two passes)
PHASE 2 Autonomous Operations Planned
  • OpenFang daemon replacement OpenFang NUC
  • CLI-Anything — agent-native tooling CLI-Anything
  • Self-healing escalation (auto-remediate, escalate novel issues only)
  • Multi-device orchestration NUC Mac Mini MacBook Pro
PHASE 3 Visitor Tanks Planned
