Mac Mini Setup

Run these commands in Terminal on the Mac Mini. Each step should complete before moving to the next.

Step 1: Install Homebrew

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Follow the prompts. After install, run the commands it tells you to add Homebrew to your PATH.

Step 2: Install Docker (via Colima)

Docker Desktop requires macOS Ventura 13.3+. Since the Mac Mini runs Monterey 12.7.6, we use Colima — a lightweight open-source Docker runtime that outperforms Docker Desktop.

brew install docker docker-compose colima

Start Colima with 8GB RAM and 4 CPUs allocated:

colima start --cpu 2 --memory 8 --disk 60

Verify Docker works:

docker run hello-world

To start Colima automatically on boot:

brew services start colima

Note: Colima uses the same docker and docker compose commands as Docker Desktop. All Digiquarium scripts work unchanged.

Step 3: Install Tailscale

brew install --cask tailscale

Open Tailscale from the menu bar and sign in. This creates the encrypted mesh network.

tailscale up

Note the Tailscale IP address it assigns (e.g., 100.x.x.x). Share this with me.

Step 4: Install Rust (for OpenFang + Rustunnel)

curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

Choose "1) Proceed with standard installation" when prompted. Then:

source "$HOME/.cargo/env"
rustc --version

Step 5: Install Node.js 22 (for GuardianAgent)

brew install node@22

Step 6: Clone the Digiquarium repo

cd ~
git clone https://github.com/ijnebzor/thedigiquarium.git digiquarium
cd digiquarium

Step 7: Copy the .env file

The .env file contains API keys and isn't in git. Copy it from the NUC via Tailscale:

# On the Mac Mini (after both devices are on Tailscale):
scp ijneb@[NUC_TAILSCALE_IP]:/home/ijneb/digiquarium/.env ~/digiquarium/.env

Or create it manually with the same API keys.

Step 8: Verify everything

docker run hello-world    # Should print "Hello from Docker!"
colima status             # Should show "Running"
rustc --version           # Should show 1.94+
node --version            # Should show v22.x
tailscale status          # Should show connected
ls ~/digiquarium/.env     # Should exist

MacBook Pro 2018 Setup

The MacBook Pro is the monitoring station. Lighter setup.

Step 1: Install Tailscale

brew install --cask tailscale
tailscale up

This connects the MacBook to the same mesh network as the Mac Mini and NUC.

Step 2: Install Claude Desktop

Download from claude.ai/download. This will be the management interface for the entire fleet via MCP.

NUC Setup (Tailscale)

The NUC needs Tailscale to join the mesh. I'll handle this remotely once you confirm the Mac Mini is on Tailscale.

Install Tailscale on WSL2

curl -fsSL https://tailscale.com/install.sh | sh
sudo tailscale up

Note: sudo is slow on this WSL2 instance. May need to be run from the Windows host instead.

After All Devices Are Connected

What happens next (I handle this remotely)

Once all three devices are on Tailscale, I will:

1. Deploy the Digiquarium MCP server on the Mac Mini

2. Build OpenFang (Rust compile, ~15 min on Mac Mini)

3. Build Rustunnel (Rust compile)

4. Transfer tank data (brain.md, soul.md, baselines) from NUC → Mac Mini

5. Start tanks on Mac Mini, verify they resume correctly

6. Configure cross-device MCP management

7. Run security audit on the expanded fleet
