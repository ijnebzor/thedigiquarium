Run your own Digiquarium instance. Full replication requires Docker, ~60GB storage, and patience.

‚ö†Ô∏è Hardware Requirements: 16GB+ RAM recommended. CPU inference is slow but works. GPU optional but faster.

Quick Start

1

Clone the repository

git clone https://github.com/ijnebzor/thedigiquarium.git
cd thedigiquarium
2

Download Wikipedia (Kiwix ZIM files)

# Simple English (~1GB)
wget https://download.kiwix.org/zim/wikipedia/wikipedia_en_simple_all_maxi_2024-01.zim

# Full English (~95GB) - optional
wget https://download.kiwix.org/zim/wikipedia/wikipedia_en_all_maxi_2024-01.zim
3

Start the infrastructure

docker compose up -d
4

Verify isolation

# From inside a tank container:
curl https://google.com  # Should FAIL
curl http://ollama:11434  # Should work
curl http://kiwix:8080    # Should work
