Replicate the Experiment

This guide will help you set up an identical Digiquarium environment on your own hardware. The process is automated and should take approximately 30-60 minutes depending on your internet connection (Wikipedia downloads are large).

System Requirements

  • RAM: 16GB minimum (32GB recommended for all 17 tanks)
  • Storage: 100GB minimum (Wikipedia ZIM files are ~50GB)
  • CPU: 4+ cores recommended
  • OS: Linux (Ubuntu 22.04+), macOS, or Windows with WSL2
  • Docker: Docker Engine 24+ and Docker Compose

Quick Start (One Command)

For those who want to get started immediately:

curl -fsSL https://raw.githubusercontent.com/ijnebzor/thedigiquarium/main/setup/install.sh | bash

This will clone the repository, install dependencies, download Wikipedia, and start the experiment.

Manual Setup (Step by Step)

1 Clone the Repository

git clone https://github.com/ijnebzor/thedigiquarium.git
cd thedigiquarium

2 Install Ollama

Ollama provides local LLM inference:

# Linux/macOS
curl -fsSL https://ollama.com/install.sh | sh

# Pull the model
ollama pull llama3.2:latest

3 Download Wikipedia

Download the offline Wikipedia files (this takes a while):

./setup/download-wikipedia.sh

Downloads: Simple English (~1GB), Full English (~20GB), and language variants.

4 Configure Environment

Copy the example environment file and adjust if needed:

cp .env.example .env
# Edit .env to set OLLAMA_HOST if not localhost

5 Start the Experiment

# Start all tanks
docker compose up -d

# Check status
docker compose ps

# View logs
docker compose logs -f tank-01-adam

6 Access the Dashboard

Open the local dashboard to observe your specimens:

# Serve the dashboard locally
python3 -m http.server 8080 --directory website/dashboard

# Open in browser
open http://localhost:8080

Download Scripts

install.sh download-wikipedia.sh

Verifying Your Setup

After starting, verify everything is working:

# Check all containers are running
docker ps | grep tank

# Check Ollama connectivity
curl http://localhost:11434/api/tags

# Check Wikipedia is accessible
curl http://localhost:8888/wikipedia_en_simple_all/A/Main_Page

# Run a baseline assessment
docker exec tank-01-adam python3 /tank/baseline.py

Customizing Your Experiment

You can modify the experiment by editing docker-compose.yml. Each tank is configured with environment variables for language, gender framing, and Wikipedia source.
