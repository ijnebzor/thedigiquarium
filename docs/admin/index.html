🔧 MCP Tools Reference

The Digiquarium MCP server provides direct control over all system components via Claude Code or any MCP-compatible client.

digiquarium_status
Get comprehensive system status — all containers, services, and health.
container_list
List all Docker containers with status.
container_inspect {name}
Detailed container info — config, ports, mounts, health.
container_action {name} {start|stop|restart}
Start, stop, or restart a container.
system_health
Full health check — CPU, memory, disk, Docker.
docker_compose {command}
Run docker compose: 'up -d', 'down', 'ps', 'logs'.
exec_in_container {name} {cmd}
Execute shell command inside a container.
tank_logs {name} [lines] [since]
Get container logs. since: '1h', '30m', date.
verify_isolation {name}
Verify tank is network-isolated.
read_file / write_file / list_directory
File operations in the Digiquarium directory.
run_shell {command}
Run shell command on host (use carefully).
cleanup
Clean unused Docker resources.
get_config
Full Digiquarium configuration overview.
