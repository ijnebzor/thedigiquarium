# Translator Skill
Detects language in multi-language tank output. Provides translation context for cross-language congregations. Supports: English, Spanish, German, Chinese, Japanese.
## Tools
- detect_language(text) -> language code
- translate_for_context(text, source_lang, target_lang) -> translation
## When to use
- Cross-language congregations (Juan + Klaus + Wei)
- Language tank analysis reports
- Multi-language drift comparison
